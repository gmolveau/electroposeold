BloxHover - jQuery Image Hover Effects Plugin



1) Installation

Include latest jQuery version and BloxHover plugin in the head secion of your document. You can also include easing library for smoother animations, but it's not required, plugin can work without it.

                <head>
                    <script src="http://ajax.googleapis.com/ajax/libs/jquery/2.0.3/jquery.min.js"></script> 
                    <script src="http://cdnjs.cloudflare.com/ajax/libs/jquery-easing/1.3/jquery.easing.min.js"></script> 
                    <script src="bloxhover.jquery.min.js"></script> 
                </head>




2) HTML Structure and Usage

First create HTML code like this:

                    <div class="mydiv">
                        <img src="image.jpg"/>
                        <div>Stuff that you put in this div will be shown in hover! Any HTML Code can go here</div>
                    </div>

Then, all we have to do is to call BloxHover. It should be called on the div that wraps the image and inner content, like this:

                    <script> 
                        $(document).ready(function() {
                            $('.mydiv').bloxhover();
                        });
                    </script>

And now, BloxHover effect will work on all .mydiv elements that you have on your page.





3) Parameters

BloxHover can be configured with several parameters. You can set them when you call the BloxHover plugin like this: 
                    <script> 
                        $(document).ready(function() {
                            $('.mydiv').bloxhover({delay: 100});
                        });
                    </script>


All Options

Here you can see all possible parameters that BloxHover can take:

<script> 
    $(document).ready(function() {
        $('.mydiv').bloxhover({
            effect: "vertical", // accepted strings: 'square', 'square reveal', 'vertical', 'vertical reveal', 'vertical alternate', 'vertical alternate reveal', 'horizontal', 'horizontal reveal', 'horizontal alternate', 'horizontal alternate reveal'
            sliceCount: 8, // the number of slices 
            color: 'rgba(0, 0, 0, 0.5)', //rgba color of the slices
            duration: 300, //how long should the animation of each slice last
            delay: 100 // delay between slice animations
        });
    });
</script>
